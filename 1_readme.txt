Modèle REMEPP - Version 4.0
================================

Auteurs du modèle REMEPP :

Description : Le modèle REMEPP simule le remplissage de la plage en tant qu’espace de pratique de tourisme et de loisir

Plateforme : Le modèle REMEPP est développé sous la plateforme GAMA plateforme (http://gama-platform.org/), version 1.7.0. Merci d'utiliser cette version de GAMA pour charger le modèle et lancer les simulations.
--> GAMA 1.7.0 est disponible ici : https://ao.univ-lr.fr/index.php/s/oG2cmHYqD5QbT5S

Matériel : Le modèle REMEPP est constitué du code gaml (fichier REMEPP_va.gaml) et de plusieurs fichiers shp (inclus dans le répertoire du modèle). 
--> l'ensemble est disponible ici : https://ao.univ-lr.fr/index.php/s/s48AxiJEa2Ko2Ag



Installation de GAMA et chargement du modèle
=============================================

1. Installer le logiciel GAMA 1.7.0

2. Vérifier que la version de Java est à jour : version compatible, 8 Update 201 (15 janvier 2019). Sinon mettre à jours ici : https://www.java.com/fr/download/manual.jsp

3. Télécharger et installer le Java SE Development Kit (JDK) correspondant à la version de Java (8u201). Lien de téléchargement : https://www.oracle.com/technetwork/java/javase/downloads/index-jsp-138363.html#javasejdk

4. Lancer GAMA et choisir l’emplacement du « Workspace » puis cliquer sur ok
pour intégrer le modèle à GAMA :
- Copier le fichier zippé du modèle sur son ordinateur puis le dé-zipper quelque part
- dans la fenêtre « Models » clic droit sur « User Model » puis « Import » et « Existing Projects Into Workspace »
- dans la fenêtre qui s’ouvre, aller chercher le dossier dé-zipper du modèle et sélectionner
« Copy Projects into Workspace » et cliquer sur « Finish »
